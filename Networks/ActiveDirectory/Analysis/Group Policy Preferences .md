# SYSVOL 
is the domain-wide share in Active Directory to which all authenticated users have read access. SYSVOL contains logon scripts, group policy data, and other domain-wide data which needs to be available anywhere there is a Domain Controller (since SYSVOL is automatically synchronized and shared among all Domain Controllers).

#### Note: Sometimes you can find shares the replicate the sysvol share and you can access it as anonymous
